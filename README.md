# Rails Minimal API
## How to create this repository
```
$ rails new rails-minimal-api --minimal --api -T
$ vim Dockerfile
$ vim docker-compose.yml
```
## How to use
First, git clone, and docker-compose up.  
```
$ docker-compose up
```  
Then, you can open app on your browser or curl.
```
$ curl http://localhost:3001
```
